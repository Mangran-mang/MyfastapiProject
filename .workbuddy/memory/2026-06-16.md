# 2026-06-16 工作日志

## 前端移动端响应式适配
- 为整个前端项目（Vue 3 + Vite）添加移动端响应式适配
- 断点：768px（iPad/手机分界线）

### 修改文件清单
1. **main.css** - 添加全局移动端断点 @media (max-width: 768px)：
   - `.container` max-width 从固定 800px 改为移动端 100%
   - `.card` padding 缩小
   - input/textarea/select 字体改为 16px（防 iOS 自动缩放）
   - h2/h3 字号缩小
2. **App.vue** - 内容区 padding 改为通过 `.main-content` class 控制，移动端减小
3. **AppHeader.vue** - 最大改动：
   - 桌面端导航加 `.nav-desktop` class，移动端隐藏
   - 新增汉堡菜单按钮（三横线 → X 动画）
   - 新增移动端下拉菜单（`<Transition>` 动画）
   - 菜单项点击自动关闭
4. **PostCard.vue** - post-header/meta 添加 flex-wrap，字号缩小
5. **LoginView.vue / RegisterView.vue** - 移动端 padding-top 减小，卡片自适应
6. **PostListView.vue** - 板块选择器和分页尺寸适配
7. **PostDetailView.vue** - meta 换行、actions 换行、楼中楼缩进减小、comment-header 换行
8. **CreatePostView.vue** - textarea 最小高度缩小
9. **ProfileView.vue** - 信息行改为纵向排列（移动端）
10. **NotificationsView.vue** - notif-header 换行、内容自动换行
