# 2026-06-14 工作日志

## 创建 Vue 3 前端项目

### 项目结构
- `frontend/` - Vite + Vue 3 前端项目
- SPA 架构，Hash 路由，开发时代理到后端

### 完成内容
1. **项目脚手架**: package.json, vite.config.js (配了 /api 代理), index.html, main.js
2. **API 封装**: `src/api/index.js` - 统一请求方法，自动 token 刷新，覆盖全部后端接口
3. **路由**: 7 个页面（登录、注册、帖子列表、帖子详情、发帖、个人主页、通知）
4. **视图组件**:
   - LoginView - 邮箱密码登录
   - RegisterView - 注册新用户
   - PostListView - 帖子列表（分页、板块筛选）
   - PostDetailView - 帖子详情 + 评论（含楼中楼）+ 点赞/收藏
   - CreatePostView - 发帖（选板块）
   - ProfileView - 个人信息 + 我的帖子
   - NotificationsView - 通知列表 + 已读/未读
5. **共享组件**: AppHeader（导航栏、未读通知数轮询）、PostCard（帖子卡片）
6. **构建验证**: `npm run build` 成功，产出 117KB JS + 6KB CSS
